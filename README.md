# NYL × Girls Who Code Workshop

**A Day in the Life of a Software Engineer** — Interactive workshop activity for the New York Life × Girls Who Code event.

## What It Does

Partecipents work through 4 levels on their phones:

1. **Build** — Drag-and-drop JavaScript code blocks to collect applicant info
2. **AI Review** — Spot an AI hallucination (fake API endpoint)
3. **Build** — Arrange code to calculate premium and submit to server
4. **AI Review** — Find a security hole (SSN logged to console)
